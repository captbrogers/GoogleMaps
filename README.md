<h1 align="center">GoogleMaps Plugin</h1>
<h3 align="center">for HorizontCMS</h3>

<hr>


## Install

Download and copy into /plugins/GoogleMaps folder. On the admin panel go to Themes&Apps/Plugins. Click on install and activate.

## Shortcode
``` {[GoogleMaps]} ```
